# ST Timelines
Tools for use with ST: Timelines

## match_crew.py

Determines possible crew members for Boss Battles. The data is loaded from Datacore's `crew.json`.

For each slot the possible trait combinations that have results are listed, along with the possible crew members. There is also a list of those crew members who match more than once, with both the slot and the trait combinations.

It is possible to set the maximum rarity (star level) for the crew to be included, and exclude specific crew members eg because they have already been attempted. When an excluded crew member would be matched for a slot then the trait combination is discarded from the results.
